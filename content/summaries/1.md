Session 1 covers command-line coding agents and how they differ from the
online chat window. The speaker demos Claude Code: what it can do to your local
files, the security tradeoffs, CLAUDE.md, permissions, models, context, skills,
and reviewing every change as a diff.

* [Command-line agents vs online chat](https://youtu.be/gOUW4jmGJVY?t=223) -- how a launched-from-the-terminal agent differs from chat GPT-style web tools.
* [Launching Claude Code and local file access](https://youtu.be/gOUW4jmGJVY?t=285) -- a local program that reads your files and runs commands, unlike a website.
* [Prompt injection and security worries](https://youtu.be/gOUW4jmGJVY?t=414) -- agents act on text they read, so poisoned instructions are a real risk.
* [No privacy: prompts go to remote servers](https://youtu.be/gOUW4jmGJVY?t=606) -- assume anything submitted is used for training; do not feed it confidential work.
* [CLAUDE.md: project rules in markdown](https://youtu.be/gOUW4jmGJVY?t=918) -- a file the agent reads on every run, like instructions for an aggressive TA.
* [Permissions: the agent asks first](https://youtu.be/gOUW4jmGJVY?t=1420) -- approving the commands and file reads the agent wants to run.
* [Permission modes via shift+tab](https://youtu.be/gOUW4jmGJVY?t=1578) -- cycling default, accept-edits, plan, and auto mode, and the risks of each.
* [Reviewing a generated graphics lecture](https://youtu.be/gOUW4jmGJVY?t=1894) -- a full slide deck created from a prompt, shown as a reviewable diff.
* [git and diffs as the safety net](https://youtu.be/gOUW4jmGJVY?t=2340) -- using version control and IntelliJ diff views to keep control of changes.
* [Managing context: clear and compact](https://youtu.be/gOUW4jmGJVY?t=2559) -- the context window fills as you go; /clear resets and /compact summarizes.
* [Skepticism about skills](https://youtu.be/gOUW4jmGJVY?t=2655) -- skills are just markdown instructions; CLAUDE.md often does the same job.
* [Demo: explain an unfamiliar repo](https://youtu.be/gOUW4jmGJVY?t=3158) -- pointing a fresh agent at the course repo to summarize its structure.
