Carson walks through keeping all course materials in plain-text markdown so agents can read and edit them directly. He covers Slidev for slide decks, mermaid diagrams, converting old decks, fact-checking, and PDF export.

* [Why markdown and agents love plain text](https://youtu.be/5i7lhEwNf_Q?t=0) -- everything in the repo (lectures, quizzes, notes) is markdown
* [Agents can dump all the text into context](https://youtu.be/5i7lhEwNf_Q?t=125) -- contrast with the old Google Slides workflow he paid a student to convert
* [Text diffs make changes easy to review](https://youtu.be/5i7lhEwNf_Q?t=221) -- the IntelliJ diff view shows exactly what an agent changed
* [Slidev intro and features](https://youtu.be/5i7lhEwNf_Q?t=315) -- free markdown slide system at sli.dev, code-friendly with syntax highlighting
* [Progress-bar plugin built by Claude](https://youtu.be/5i7lhEwNf_Q?t=503) -- a quality-of-life Slidev plugin he had Claude write
* [The Slidev workflow](https://youtu.be/5i7lhEwNf_Q?t=596) -- run slidev from the command line pointed at a lecture, present on localhost
* [Mermaid diagrams baked into Slidev](https://youtu.be/5i7lhEwNf_Q?t=816) -- flowcharts, sequence and state diagrams as text, no drawing tool
* [Converting old Google Slides or PowerPoint decks](https://youtu.be/5i7lhEwNf_Q?t=1066) -- Claude reads those formats and converts to Slidev
* [Screenshot to slide live demo](https://youtu.be/5i7lhEwNf_Q?t=1129) -- paste an image and Claude adds a matching mermaid slide
* [Consistent style with CLAUDE.md](https://youtu.be/5i7lhEwNf_Q?t=1285) -- discipline the voice, still rewrite to keep it your own
* [Curriculum gap analysis against peer schools](https://youtu.be/5i7lhEwNf_Q?t=1442) -- compare a class to Berkeley, MIT, or a top-10 syllabus
* [Fact-checking generated content](https://youtu.be/5i7lhEwNf_Q?t=1662) -- have the agent verify truth claims via web search
* [PDF export with just](https://youtu.be/5i7lhEwNf_Q?t=1754) -- just gen-pdfs renders every deck to PDF for Canvas
