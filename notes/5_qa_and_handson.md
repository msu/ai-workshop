# Q&A and Hands-On

## Your turn at the keyboard

---

## The Tension

* The same agent is a patient TA and a tireless cheat
* It can explain a concept or write the whole assignment
* The fix is structural, not technological

::right::

```
  good: explain pointers to me
  bad:  write my malloc for me

  same tool, your call
```

---

## AI and Non-AI Tracks

* Offer a final project in two tracks
* Non-AI track is graded for the learning, not the output
* Let students choose, and make the choice honest

::right::

```
  Track A: no AI
    smaller scope, graded on craft

  Track B: AI allowed
    bigger scope, graded on result
```

---

## You Have to Write the Code

* You cannot read code well until you have written it
* Outsourcing the writing skips the learning
* The classroom is the safe place to build that skill

::right::

```
  write  -> struggle -> understand
        -> read others' code
        -> build real systems

  skip the first step and the
  rest falls apart
```

## Summary

* The agent is both a good tutor and an easy cheat
* The answer is structural: how you assess, not what you ban
* AI and non-AI tracks let students opt into real work
* Writing code is the prerequisite for reading code
* The classroom is the safe place to build that skill
* Try the tools yourself before you decide your policy
