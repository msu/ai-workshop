Effective AI for Educators

A hands-on workshop on using command-line AI coding agents to author course materials, build assessments, and run a course. Recordings, notes, and transcripts for each session are collected below.

TL;DR - (click to expand)

Big Ideas

  • Keep course material as plain text so an agent can read and edit all of it.
  • Point a command-line agent at your files, then review every change as a diff.
  • Treat CLAUDE.md as standing rules, like instructions for an aggressive TA.
  • Assume prompts are not private, and watch for prompt injection.
  • Use AI for prep and grading glue; move assessment in-person to protect learning.
  • The university is a safe space to learn fundamentals and AI at the same time.

Best Tricks

  • Standing project rules in CLAUDE.md so you stop re-explaining yourself (15:18)
  • Screenshot straight into a slide, and convert old PowerPoint/Google decks (18:49)

  • Draft quiz questions from lecture notes, then generate the answer key and TA rubric (6:41)

  • GitHub Actions autograder for homework (33:43)
  • Canvas final grades from a CSV, plus Tamper Monkey scripts to kill Canvas friction (3:14)
  • One-page podium notes generated from the lecture deck (1:07)
